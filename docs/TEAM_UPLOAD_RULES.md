# Team Upload Rules

## Character sprites

Upload character sprites to Cloudinary with simple filenames.

Examples:

```txt
ax_idle1.png
ax_idle2.png
ax_walk1.png
ax_jump1.png
pura_idle1.png
owl_walk01.png
unicorn_jump1.png
```

Do not use random long filenames or names with spaces.

## GitHub

GitHub is for the game files only. Large sprite folders stay on Cloudinary.
