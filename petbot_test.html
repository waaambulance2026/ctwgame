PetBot Tamagotchi setup

Simple version. Only 3 buttons matter: A = left heart, B = pacifier, C = right heart.

Only 3 press areas

These are just the invisible press areas. Move only these if needed.

If you are unsure, press Reset 3 buttons and leave these alone.

LCD screen fit

Drag the blue LCD box on the toy. Drag the bottom-right corner to resize it. Use the button below to snap it back to the cut-out screen.

Screen style: colour screen only.

Button map

A = previous
B = select/menu
C = next
